📋 ÉTAPES À SUIVRE (10 minutes)

1️⃣  SUPABASE
   □ Créer compte sur https://supabase.com
   □ Créer un nouveau projet
   □ Exécuter DONNEES_EXEMPLE.sql dans SQL Editor
   □ Créer utilisateur dans Authentication > Users
   □ Copier URL et anon key depuis Settings > API

2️⃣  CONFIGURATION
   □ Ouvrir: src/environments/environment.ts
   □ Remplacer VOTRE_URL_SUPABASE par l'URL copiée
   □ Remplacer VOTRE_ANON_KEY_SUPABASE par la clé copiée
   □ Sauvegarder le fichier

3️⃣  DÉMARRAGE
   □ Terminal: npm start
   □ Ouvrir: http://localhost:4200
   □ Se connecter avec email/mot de passe Supabase
